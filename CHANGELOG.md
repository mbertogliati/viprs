### Features

- Demand-driven, horizontally-threaded image processing pipeline
- Monomorphized operation dispatch (zero `dyn Trait` on hot paths)
- Zero heap allocations in pixel-path code
- Arithmetic operations: invert, linear, add, multiply, abs, sign, round, floor, ceil, bandmean, boolean (and/or/eor)
- Colour operations: colourspace conversions (sRGB, Lab, XYZ, HSV, LCh, scRGB, CMC, GREY16, YXY)
- Convolution: gaussian blur, sharpen, unsharp mask, frequency filtering (FFT)
- Resample: resize, shrink, zoom, thumbnail with auto-rotation
- Perceptual enhance composite pipeline
- Codec support: JPEG, PNG, WebP, TIFF, GIF, HEIF/AVIF, EXR (feature-gated)
- Thread-pool based tile scheduler with work-stealing
- Pipeline arena for complex multi-operation graphs
- Ergonomic `ImageApi` for common workflows
- Streaming support for web-service workloads
